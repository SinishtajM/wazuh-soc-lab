## Sanitization

The published PNG files contain flattened redactions over private addresses, usernames, and environment-specific identifiers. The published files were also checked to confirm that PNG text, XMP, EXIF, and comment metadata chunks are absent.

The repository audit workflow rejects future screenshots containing `tEXt`, `zTXt`, `iTXt`/XMP, or `eXIf` chunks.

Before replacing any screenshot, review it for:

- Internal and public IP addresses
- Usernames and real names
- Hostnames and browser URLs
- Email addresses
- Tokens, keys, credentials, and certificate material
- Sensitive paths or command output
- Embedded EXIF, XMP, comments, or editor metadata
